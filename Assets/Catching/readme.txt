game씬을 추가해서 Display1, Display2를 만들고 해상도를 1920x1080으로 설정합니다.
Display1은 인형뽑기 안에서 top view 시점이며 게임은 display1에서 진행(마우스는 display1에 있음)
display2는 실제 사람의 시야로 세팅. (유저가 바라보는 모니터)

테이블에 세워져 있는 모니터가 display2가 되며, 테이블에 눕혀서 설치되는 터치 모니터가 display1이 됩니다.
유저는 display2만 바라보면서 게임 진행.

게임 시작 3초간 준비를 마치고 game start.
5초간 터치팬을 쥐고 자유롭게 움직인 뒤, 로봇팔이 내려가면서 인형뽑기 연출.
